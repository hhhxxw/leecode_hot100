# 思路
定义一个ans，初始化很大一个数，然后直接遍历数组， 如果ans 小于当前的数，就将其赋值给ans，继续循环，直到循环结束。
```java
class Solution {

    public int findMin(int[] nums) {

        int ans = 5000;

        for(int i = 0; i < nums.length; i ++){

            if(nums[i] < ans){

                ans = nums[i];

            }

        }

        return ans;

    }

}
```
