# 题目概述

给定一个只包括 `'('`, `')'`, `'{'`, `'}'`, `'['`, `']'` 的字符串 `s`，判断字符串是否有效。

**有效字符串需满足：**

1. 左括号必须用相同类型的右括号闭合。
2. 左括号必须以正确的顺序闭合。
3. 每个右括号都有一个对应的相同类型的左括号。

**示例：**

- `"()"` -> `true`
- `"()[]{}"` -> `true`
- `"(]"` -> `false`
- `"([)]"` -> `false`

---

# 解题思路

这道题是 **栈 (Stack)** 的经典应用场景。

**核心思想：**  
利用栈的 **后进先出 (LIFO)** 特性。当遇到左括号时，说明后面需要一个对应的右括号来闭合，我们将这个期待的右括号压入栈中。当遇到右括号时，我们检查它是否与栈顶元素（即最近期待的那个右括号）匹配。

**详细步骤：**

1. 创建一个空栈。
2. 遍历字符串中的每一个字符：

- 如果是左括号 `(`, `{`, `[`：

- 为了方便后续比较，我们不把左括号压入栈，而是 **直接把对应的右括号** `)`, `}`, `]` 压入栈。

- 如果是右括号：

- 检查栈是否为空（说明有多余的右括号）。
- 弹出栈顶元素，检查栈顶元素是否等于当前字符。
- 如果不相等（说明括号类型不匹配），直接返回 `false`。

3. 遍历结束后，检查栈是否为空：

- 如果为空，说明所有左括号都正确闭合了，返回 `true`。
- 如果不为空，说明有多余的左括号未闭合，返回 `false`。

# Java 代码

```
import java.util.Stack;

class Solution {
    public boolean isValid(String s) {
        // 剪枝：如果字符串长度是奇数，肯定无法完全闭合
        if (s.length() % 2 != 0) {
            return false;
        }

        Stack<Character> stack = new Stack<>();

        for (char c : s.toCharArray()) {
            if (c == '(') {
                stack.push(')'); // 遇到左括号，把期待的右括号入栈
            } else if (c == '{') {
                stack.push('}');
            } else if (c == '[') {
                stack.push(']');
            } else {
                // 如果是右括号
                // 1. 栈为空：说明右括号多了，没有对应的左括号
                // 2. 栈顶元素不等于当前字符：说明左右括号不匹配
                if (stack.isEmpty() || stack.pop() != c) {
                    return false;
                }
            }
        }

        // 最后栈必须为空，才说明所有左括号都闭合了
        return stack.isEmpty();
    }
}
```
