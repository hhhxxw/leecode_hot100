# 思路

先将数组进行排序，这样相同的元素一定是相邻的，然后使用双指针算法，两个元素两个元素进行比对，如果相同则再比较下一对，如果不同则直接第一个元素即可。
```java
class Solution {
    public int singleNumber(int[] nums) {
        Arrays.sort(nums);
        if(nums.length == 1){
            return nums[0];
        }
        int i = 0, j = 1;
        while(j < nums.length){
            if(nums[i] == nums[j]){
                i += 2;
                j = i + 1;
            }else{
                return nums[i];
            }
        }
        return nums[i];
    }
}
```
