这道题目我一开始没有思路，然后在题解中看到一句话直接就悟了，最大利润 = 今天利润 - 前面n天的最小值

```java
class Solution {
    public int maxProfit(int[] prices) {
        // 最大利润，等于今天- 前面n天的最小值
        int maxValue = Integer.MIN_VALUE;
        int last_min_value = Integer.MAX_VALUE;
        for(int i = 0; i < prices.length; i ++){
            if(prices[i] < last_min_value){
                last_min_value = prices[i];
            }
            int currentValue = prices[i] - last_min_value;
            if(currentValue > maxValue){
                maxValue = currentValue;
            }
        }
        if(maxValue < 0){
            return 0;
        }else{
            return maxValue;
        }
    }
}
```
