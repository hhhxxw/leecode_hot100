# 题目概述

给你一个整数数组 `nums`，判断是否存在三个元素 `a, b, c` 使得 `a + b + c = 0` ？请你找出所有和为 `0` 且 **不重复** 的三元组。

**示例：**

- 输入：`nums = [-1, 0, 1, 2, -1, -4]`
- 输出：`[[-1, -1, 2], [-1, 0, 1]]`

---

# 解题思路

暴力法是三层循环，复杂度 ![](https://cdn.nlark.com/yuque/__latex/a77815a52a2d0050828079fb142b1410.svg)，肯定会超时。我们可以利用 **排序 + 双指针** 将复杂度降低到 ![](https://cdn.nlark.com/yuque/__latex/f2d5f588234eb61a559ff90c41511b85.svg)。

**核心步骤：**

1. **排序**：首先对数组进行升序排序，这对于后续去重和移动指针非常重要。
2. **固定第一个数**：遍历数组，用 `nums[i]` 作为第一个数。

- **剪枝**：如果 `nums[i] > 0`，因为数组已经排序，后面不可能有负数来抵消它，所以直接结束。
- **去重**：如果 `nums[i] == nums[i-1]`，说明这个数已经作为第一个数处理过了，为了避免重复结果，直接 `continue` 跳过。

3. **双指针寻找另外两个数**：

- 设 `left = i + 1`，`right = n - 1`。
- 计算 `sum = nums[i] + nums[left] + nums[right]`。
- 如果 `sum == 0`：

- 找到一组解，加入结果集。
- **去重**：`while (left < right && nums[left] == nums[left+1]) left++;`
- **去重**：`while (left < right && nums[right] == nums[right-1]) right--;`
- 同时移动 `left++` 和 `right--`。

- 如果 `sum < 0`：说明和太小了，需要变大，`left++`。
- 如果 `sum > 0`：说明和太大了，需要变小，`right--`。

# 代码实现

```java
import java.util.ArrayList;
import java.util.Arrays;
import java.util.List;

class Solution {
    public List<List<Integer>> threeSum(int[] nums) {
        List<List<Integer>> result = new ArrayList<>();
        if (nums == null || nums.length < 3) {
            return result;
        }

        // 1. 排序
        Arrays.sort(nums);

        // 2. 遍历第一个数
        for (int i = 0; i < nums.length; i++) {
            // 剪枝：如果第一个数已经大于0，后面不可能凑出0了
            if (nums[i] > 0) {
                break;
            }

            // 去重：如果当前数和前一个数相同，跳过
            if (i > 0 && nums[i] == nums[i - 1]) {
                continue;
            }

            int left = i + 1;
            int right = nums.length - 1;

            while (left < right) {
                int sum = nums[i] + nums[left] + nums[right];
                
                if (sum == 0) {
                    // 找到一组解
                    result.add(Arrays.asList(nums[i], nums[left], nums[right]));
                    
                    // 去重：跳过重复的 left
                    while (left < right && nums[left] == nums[left + 1]) {
                        left++;
                    }
                    // 去重：跳过重复的 right
                    while (left < right && nums[right] == nums[right - 1]) {
                        right--;
                    }
                    
                    // 找到答案后，双指针同时收缩
                    left++;
                    right--;
                } else if (sum < 0) {
                    // 和太小，left 右移变大
                    left++;
                } else {
                    // 和太大，right 左移变小
                    right--;
                }
            }
        }
        return result;
    }
}
```

```python
class Solution:
    def threeSum(self,  nums: list[int]) -> list[list[int]]:
        res = []
        n = len(nums)
        if n < 3:
            return res
        nums.sort()
        for i in range(n):
            if nums[i] > 0:
                break
            # 对 i 进行去重
            if i > 0 and nums[i] == nums[i - 1]:
                continue

            left = i + 1
            right = n - 1

            while left < right:
                s = nums[i] + nums[left] + nums[right]
                if s == 0:
                    # 拼接数组
                    res.append([nums[i], nums[left], nums[right]])
                    while left < right and nums[left] == nums[left + 1]:
                        left += 1
                    while left < right and nums[right] == nums[right - 1]:
                        right -= 1
                    left += 1
                    right -= 1
                elif s > 0 :
                    right -= 1
                else:
                    left += 1
        return res
```
