# 题目概述

给定一个长度为 `n` 的整数数组 `height`。有 `n` 条垂线，第 `i` 条线的两个端点是 `(i, 0)` 和 `(i, height[i])`。  
找出其中的两条线，使得它们与 `x` 轴共同构成的容器可以容纳最多的水。返回容器可以储存的最大水量。

**注意：** 你不能倾斜容器。**核心公式：** `Area = min(height[left], height[right]) * (right - left)`

---

# 解题思路

这道题是 **双指针 (Two Pointers)** 的经典题目。如果我们用暴力法（两层循环），复杂度是 ![](https://cdn.nlark.com/yuque/__latex/f2d5f588234eb61a559ff90c41511b85.svg)，会超时。我们可以利用双指针将复杂度降为 ![](https://cdn.nlark.com/yuque/__latex/e65a67ac353abeeff44c359310d05c02.svg)

**核心逻辑：**

1. **初始化**：设置两个指针 `left = 0` 和 `right = n - 1`，分别指向数组的最左端和最右端。此时容器的宽度最大。
2. **计算面积**：当前的面积为 `min(height[left], height[right]) * (right - left)`。
3. **移动指针**：

- **关键点：我们应该移动哪根柱子？**
- 如果移动较高的那根柱子，宽度变小了，而新的高度受限于较短的那根柱子，所以面积一定只会变小，不可能变大。
- 因此，为了寻找更大的面积，我们必须 **移动较短的那根柱子**，试图找到一根更高的柱子来替代它，从而可能增加高度（虽然宽度减小了，但高度增加可能带来更大的收益）。
- 如果 `height[left] < height[right]`，则 `left++`。
- 如果 `height[left] > height[right]`，则 `right--`。
- 如果相等，移动任意一个都可以（或者同时移动，但这需要更细致的证明，简单起见移动任意一个即可）。

4. **循环终止**：当 `left >= right` 时结束。

# 代码

```java
class Solution {
    public int maxArea(int[] height) {
        int left = 0;
        int right = height.length - 1;
        int maxArea = 0;

        while (left < right) {
            // 计算当前的面积：高度取两者最小值，宽度是索引之差
            int currentHeight = Math.min(height[left], height[right]);
            int currentWidth = right - left;
            int currentArea = currentHeight * currentWidth;
            
            // 更新最大面积
            maxArea = Math.max(maxArea, currentArea);

            // 移动指针策略：总是移动较短的那根柱子
            // 因为移动较高的柱子不可能让面积变大（受限于短板效应）
            if (height[left] < height[right]) {
                left++;
            } else {
                right--;
            }
        }

        return maxArea;
    }
}
```

```python
class Solution:
    def maxArea(self, height: List[int]) -> int:
        left = 0
        right = len(height) - 1
        maxArea = 0
        while left < right :
            currentWideth = right - left
            currentHeight = min(height[left], height[right])
            currentArea = currentWideth * currentHeight
            maxArea = max(maxArea, currentArea)
            if height[left] < height[right]:
                left += 1
            else:
                right -= 1
        return maxArea   
```
