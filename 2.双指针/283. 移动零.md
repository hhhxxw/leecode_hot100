# 题目概述

给定一个数组 `nums`，编写一个函数将所有 `0` 移动到数组的末尾，同时保持非零元素的相对顺序。

**要求：**

1. 必须在原数组上操作，不能拷贝额外的数组。
2. 尽量减少操作次数。

**示例：**

- 输入：`[0, 1, 0, 3, 12]`
- 输出：`[1, 3, 12, 0, 0]`

---

# 解题思路

这也是 **双指针 (Two Pointers)** 的应用。我们可以维护一个指针 `index`，用来指向 **下一个非零元素应该放置的位置**。

**步骤：**

1. **第一次遍历**：

- 遍历整个数组。
- 只要遇到非零元素 `nums[i]`，就把它放到 `nums[index]` 的位置。
- 放完之后，`index` 指针向后移动一位。
- 如果是零元素，什么都不做（跳过）。

2. **第二次遍历**（补零）：

- 当第一次遍历结束后，所有非零元素都已经按顺序挤到了数组的前面 `0` 到 `index-1` 的位置。
- 此时，从 `index` 到数组末尾 `n-1` 的位置应该全部填充为 `0`。

**优化思路（一次遍历）：**  
其实可以在第一次遍历时直接做交换操作（swap），这样就不需要第二次补零了。但这在非零元素很少时可能会增加写入次数。通常上述两遍遍历的方法更直观且写入次数较少（对于零多的情况）。

这里给出最直观的“移动非零 + 补零”方案。

---

# 代码

```java
class Solution {
    public void moveZeroes(int[] nums) {
        if (nums == null || nums.length == 0) {
            return;
        }

        // index 指针记录非零元素应该存放的位置
        int index = 0;

        // 1. 遍历数组，将所有非零元素依次挪到前面
        for (int i = 0; i < nums.length; i++) {
            if (nums[i] != 0) {
                nums[index] = nums[i];
                index++;
            }
        }

        // 2. 将剩余的位置全部补零
        for (int i = index; i < nums.length; i++) {
            nums[i] = 0;
        }
    }
}
```

```python
class Solution:
    def moveZeroes(self, nums: List[int]) -> None:
        if not nums:
            return
        index = 0
        for i in range(len(nums)):
            if nums[i] != 0:
                nums[index] = nums[i]
                index += 1
        for i in range(index, len(nums)):
            nums[i] = 0
```
